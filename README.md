# Discord Channel Activity Bumper 🚀

Простой и легковесный Discord-бот на Python, который искусственно поддерживает активность в текстовом канале. Бот с заданным интервалом отправляет невидимый символ и сразу его удаляет, привлекая внимание участников (загорается белый кружок уведомления у канала).

## Особенности
* 🔹 **Абсолютно легально:** Использует официальное Discord API (`discord.py`), что исключает риск бана аккаунта (в отличие от юзерботов).
* 🔹 **Без флуда:** Отправляет невидимый символ (Zero Width Space) и мгновенно удаляет его. История чата остается чистой.
* 🔹 **Кастомизация:** Легкая настройка интервала отправки и целевого канала.

## Быстрый старт

### 1. Подготовка в Discord
1. Создайте бота на [Discord Developer Portal](https://discord.com).
2. Во вкладке **Bot** включите тумблер **Message Content Intent**.
3. В разделе **OAuth2 -> URL Generator** выберите scope `bot` и права `Send Messages`, `Manage Messages`. Пригласите бота на сервер.
4. Включите режим разработчика в Discord и скопируйте **ID нужного текстового канала**.

### 2. Настройка скрипта
В файле `discord_up.py` заполните конфигурационные переменные:
```python
TOKEN = "ВАШ_ТОКЕН_БОТА"
CHANNEL_ID = 123456789012345678  # ID вашего канала
INTERVAL = 300  # Интервал в секундах (5 минут)
```

### 3. Запуск локально
Установите зависимости и запустите бота:
```bash
pip install -r requirements.txt
python discord_up.py
```
Скрипт полностью готов для работы 24/7 и деплоя на специализированные хостинги (например, **BotHost.ru** или аналогичные Pterodactyl-панели).
